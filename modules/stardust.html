use (self, func)
Add middleware to the stack. Middleware is called in the order it is added. While it is technically possible to add middleware after the application has started (ie, calling run), this is not supported and may lead to strange results. Also add middleware before adding routes

Parameters:

  • self stardust application
  • func function to call
run (self, ngx)
Run the application

Parameters:

  • self stardust application
  • ngx magic nginx Lua object

Usage:

    Add something like this to nginx.conf:
content_by_lua 'return require("my.stardust.module").run(ngx)';
